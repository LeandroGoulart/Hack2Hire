# Equipe - Grupo 10

## DocuSmart Seguros

Este documento registra a equipe responsavel pelo projeto **DocuSmart Seguros**, desenvolvido para o Hack2Hire - Escola da Nuvem + AWS.

O grupo trabalhou na concepcao, estruturacao, desenvolvimento e apresentacao de uma solucao para automatizar a analise inicial de documentos de sinistro usando servicos AWS, agentes e inteligencia artificial generativa.

## Frentes de Trabalho

Durante o desenvolvimento, o projeto foi organizado em frentes complementares:

| Frente | Responsabilidades |
|---|---|
| Negocio e proposta | Definir o problema, publico-alvo, beneficios e narrativa do DocuSmart Seguros. |
| Arquitetura AWS | Estruturar o fluxo com frontend, API Gateway, Lambda, Textract, Bedrock, S3, DynamoDB, IAM e CloudWatch. |
| Backend | Implementar e validar a Lambda de processamento e integracao com servicos AWS. |
| Frontend | Construir a interface demonstrativa para upload, consulta, visualizacao e exportacao de resultados. |
| Documentacao | Produzir materiais tecnicos, requisitos, arquitetura, proposta e README do projeto. |
| Apresentacao | Preparar os slides finais, roteiro de demonstracao e comunicacao para a banca. |
| Testes e demonstracao | Validar o fluxo, preparar arquivos de teste e gravar os videos de demonstracao. |

## Contribuicao Coletiva

O Grupo 10 atuou de forma colaborativa para entregar uma solucao com foco em:

- Entendimento do problema de negocio.
- Definicao da proposta para o Case C.
- Uso de servicos AWS gerenciados.
- Inclusao de IA generativa com Amazon Bedrock.
- Extracao documental com Amazon Textract.
- Orquestracao com Strands Agents.
- Persistencia de resultados em DynamoDB.
- Criacao de frontend demonstrativo.
- Preparacao da apresentacao final.
- Organizacao dos documentos uteis do projeto.
