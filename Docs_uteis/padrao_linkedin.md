## Versao principal

Participei do **Hack2Hire da Escola da Nuvem + AWS** com o **Grupo 10**, desenvolvendo o projeto **DocuSmart Seguros**, uma solucao de analise inteligente de documentos de sinistro usando arquitetura serverless, servicos AWS e inteligencia artificial generativa.

O desafio foi transformar uma rotina operacional comum em seguradoras, a analise manual de documentos, em um fluxo mais rapido, estruturado e rastreavel.

Com o DocuSmart Seguros, documentos como boletins de ocorrencia, laudos e notas fiscais podem ser processados para extrair informacoes importantes, classificar o tipo de documento e gerar uma resposta estruturada em JSON.

Principais tecnologias utilizadas:

- AWS Lambda
- Amazon API Gateway
- Amazon S3
- Amazon Textract
- Amazon Bedrock / Amazon Nova
- Amazon DynamoDB
- AWS IAM
- Amazon CloudWatch
- Strands Agents
- HTML, CSS e JavaScript no frontend demonstrativo

O projeto reforcou na pratica conceitos de:

- arquitetura serverless
- IA generativa aplicada a processos reais
- OCR e extracao documental
- integracao entre servicos AWS
- trabalho em equipe em ambiente de hackathon
- comunicacao tecnica e apresentacao de solucao
